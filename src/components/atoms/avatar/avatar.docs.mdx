## Avatar

The avatar component has two options for display. You can leave image attribute blank to show the placeholder image, or provide an image.

### Default Avatar

```jsx live
<Avatar />
```

### Avatar with custom image/props

```jsx live
<Avatar
  alt="Nate"
  size="75px"
  title="Avatar title"
  src="https://en.gravatar.com/userimage/32884912/b318719acf13489f60cba203b71e4d15.png?size=200"
/>
```
